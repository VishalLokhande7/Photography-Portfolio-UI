# 📖 Overview

Photography Portfolio UI is a responsive frontend project inspired by modern photography portfolio websites.

The project features a clean sidebar navigation, fullscreen hero image, elegant typography, responsive layout, and smooth user experience while being built entirely with HTML and CSS.

It demonstrates strong frontend fundamentals including Flexbox, responsive layouts, UI design principles, and clean code organization.
